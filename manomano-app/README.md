## Getting Started

This project use Node version >= 14.<br>

- ### Prerequisites

  As said just before, check if you have the latest recommaned version of NodeJS

  - check your current version

  ```sh
  node -v
  ```

- ### Installation

  1. Install packages

  ```sh
  yarn install
  ```

  ```sh
  yarn
  ```

  2. If you want to add more packages

  ```sh
  Add packages you want
  ```

  3. Launch project

  ```sh
  yarn start
  ```

  ```sh
  yarn start
  ```
